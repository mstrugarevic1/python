# Python Projects

Small, practical Python tools for automation, operations, and learning. Each
project is deliberately limited in scope and has its own README with examples.

| Project | Purpose | Main concepts |
| --- | --- | --- |
| [log-analyzer](log-analyzer/) | Summarize application logs | parsing, counters, CSV, CLI arguments |
| [internal-site-monitor](internal-site-monitor/) | Monitor websites and send alerts | HTTP, state, JSON, SMTP, Slack, Chart.js |
| [backup-utility](backup-utility/) | Create timestamped directory backups | paths, file copying, exclusions, dry runs |
| [git-task](git-task/) | Create Jira-based Git branches | Git command, Jira, CLI arguments, configuration |

All scripts require Python 3.10 or newer. See each project README for dependencies.
